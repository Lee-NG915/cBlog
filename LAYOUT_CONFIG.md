# PC 端布局配置说明

当前博客只面向 PC 端用户，不再维护移动端顶部栏或抽屉菜单。

## 布局结构

- 左侧固定侧边栏：`components/Sidebar.tsx`
- 右侧主内容区：`app/layout.tsx`
- 页面最小宽度：`1180px`
- 主内容最大宽度：`1180px`

## 侧边栏能力

侧边栏负责展示：

- 用户信息
- 主导航：首页、分类、关于我
- 自动生成的博客分类
- 社交链接
- 主题切换

用户信息、主导航和社交链接在 `lib/navigation.ts` 中维护。

## 分类配置

固定分类在 `lib/site.ts` 中维护：

- 技术博客
- 学习日志
- 生活随记

文章分类统计由 `lib/posts.ts` 构建期扫描 Markdown 自动生成。

## 样式原则

- 组件圆角控制在 `rounded-lg` 以内。
- 页面采用浅灰背景 + 白色内容块，深色模式使用 `#181818` / `#242424`。
- 交互以轻微位移、边框色、阴影变化为主。
- 不再为移动端断点设计专门导航或布局。
