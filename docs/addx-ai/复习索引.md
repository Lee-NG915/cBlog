## 高风险表述（先背）

- **DDD**：domain-oriented modular，非严格后端 DDD。  
- **Joy UI → Tailwind**：ADR/评估；`@mui/joy` **仍在**依赖，勿说已全量去掉。  
- **AI 修复**：Harness + 有界 loop + **PR + 人审**；默认 analyze-only；从不自动上线。  
- **搜索缓存**：以 `search-client.ts` **2min / 100 entries** 为准；旧文档 5min/50、200ms→5ms、−30% **勿当现行 SLA**。  
- **告警**：现行规范 **11** 条互斥规则。  
- **ISR Redis**：`timeoutMs:1000` + LRU；`connect()` 无超时是已知限制。  
- **Grafana SLO**：交易字段/Sentry+Loki 已落地；**OTel metrics 仍预留**，勿说 SLO 告警已全跑通。  
- **UTT conversion**：前端派发当前注释，勿说全渠道都从前端打。
