## 这间站

**给自己留的一间小书房** — 慢慢写的本子，学到哪记到哪，偶尔翻回去看一眼。

关键词：柔和 · 舒缓 · 学习 · 个人记录

## 我在意的事

- 学习可以没写完
- 工程文记语境
- 生活碎片也算数

## 你会读到

| 栏目 | 内容 |
|------|------|
| 学习记录 | 最近在啃什么、哪里还卡着 |
| 工程札记 | 写代码时的真实取舍 |
| 专题整理 | 围绕一个主题整理的复盘、问题清单和阶段性记录 |
| 生活手记 | 工作以外的小事 |

## 阅读感受

暖色、宽行距、不催人点击、按主题翻；长文有目录和轻进度提示，字体以系统中文栈优先——像笔记本，不像信息流。
