# Color 手记 · 品牌说明

> 给人看的说明在站内 [/brand](/brand)。

## 这间站是什么

**给自己留的一间小书房** — 不是资讯站，更像慢慢写的本子：学到哪记到哪，偶尔翻回去看一眼。

关键词：柔和 · 舒缓 · 学习 · 个人记录

## 写的时候在想什么

- **学习可以没写完** — 允许「还没想通」
- **工程文记语境** — 过几个月还能接上当时的自己
- **生活碎片也算数** — 不必包装，真实就好
- **页面别催人点** — 行距宽、动效轻

## 通常会记些什么

| 栏目 | 记什么 |
|------|--------|
| 学习记录 | 最近在啃什么、哪里还卡着 |
| 工程札记 | 写代码、部署时的真实取舍 |
| 生活手记 | 工作以外的小事 |

## 希望看起来怎样

- 背景偏暖，像笔记本
- 标题衬线，导航无衬线
- 滚动时轻轻浮上来
- 卡片边框淡，悬停改色就好
- 按主题翻，不搞信息流
