#### Sequence Diagram

```
sequenceDiagram
  title O2O(Sync Cart)

  actor Sales as Sales Rep
  actor C as Customer
  participant POS as POS Application(Offline)
  participant API as API Server
  participant email as Klaviyo

activate Sales

  Sales ->> C: ask email

    alt Has a Account
activate POS
        Sales ->> POS: Input customer email
activate API

        POS ->> API: getCustomerInfo
    else 
        Sales ->> POS: Add User
        POS ->> API: Create a Account
activate email
        API -) email: Trigger(Reset Password)
        email --) C: Email(Reset Password)
    end

    API -->> POS:Customer Info
    POS -->> POS:Event(changeCustomer)
    
    POS ->>+ API: fetchOnlineCart

    alt Successful case
        API -->> POS: return Order
    else resource_not_found
        POS ->> API: CreateAOnlineOrder
        POS ->> API: AssignOrderToUser
        API -->>- POS: return Order
    else others error
    POS -->> Sales: retry
    end
     

    Sales ->> POS: Searched and selected the Proudct
    Sales ->> POS: Add Proudct To Cart
    POS ->> API: AddtoCart
    Sales ->> C: If you wanna checkout
    alt wanna checkout
    Sales ->> POS: Checkout
    POS ->> API: Checkout
    POS -->> Sales: Next Order

    else
        Sales ->> POS:Push To Online
        POS ->> API:PushToOnline
            alt
                POS -->> Sales: Next Order
        deactivate Sales
            else order_not_updatable
                POS ->> API: fetchOnlineCart

                alt Successful case
                    API -->> POS: return Order
                else resource_not_found
                    POS ->> API: CreateAOnlineOrder
                    POS ->> API: AssignOrderToUser
                    API -->> POS: return Order
                end
               POS -->> Sales: retry
            end
                
    deactivate POS

    API -) email: Trigger(Cart Transferred)
    deactivate API
    email --) C : Email(Abanden Cart)

    deactivate email
    end
```
