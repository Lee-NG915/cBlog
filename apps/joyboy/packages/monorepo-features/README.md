## What is monorepo-features?

monorepo-features is a library that provides a list of features for a monorepo. It is built using React and TypeScript, it is suitable for monorepo projects and domain-driven design.

## How to identify whether a feature is applicable to feature-flags or not?

Feature Flag Applicable Scenarios:

- New feature testing (A/B testing): such as "Do users see the new UI design?".
- Progressive release (grayscale release): such as "50% of users enable the new payment method."
- Dynamic adjustment (switch control): such as "Whether to enable the recommendation system."
- Environment dependency: such as "Turn on debug logs in the test environment and turn off in the production environment."
